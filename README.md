# 天机测字神算 (Destiny Divination Web App)

这是一个结合传统八字命理（四柱八字）、《命理四书》以及汉字测字的现代网页应用程序。它能够根据用户的姓名、出生日期时间和所测汉字，提供极具神秘感和高级感的命理分析报告。

## ✨ 核心特性

- **现代暗黑琉璃设计 (Glassmorphism)**: 极具质感的 UI 设计，提供深邃、高级的视觉体验。
- **精准八字排盘**: 采用 `lunar-javascript`，根据公历生日自动推算天干地支四柱及五行属性。
- **动态交互与微动画**: 使用 `framer-motion` 打造丝滑流畅的交互与数据推演动画。
- **国学经典赋能**: 结合《滴天髓》、《渊海子平》、《三命通会》、《穷通宝鉴》四大经典命理书籍进行个性化批断。
- **响应式布局**: 完美适配桌面端和移动端，随时随地进行推演。

## 🛠️ 技术栈

- **框架**: React + TypeScript + Vite
- **UI & 动画**: 原生 Vanilla CSS + Framer Motion
- **命理引擎**: `lunar-javascript`
- **图标**: Lucide React

## 🔮 测算逻辑说明

1. **八字推演**: 将输入的公历时间转换为真太阳时/平太阳时，并转换为农历四柱。
2. **测字 Hash**: 将输入的汉字转化为哈希特征值，映射64卦象和梅花易数相关条文。
3. **四书批断**: 以八字中的“日主”（日干）为核心基点，动态拉取四大命理名著中的核心论述。
